# TOY SHOPEE – COMPLETE WEBSITE PRD

## Project Overview

Build an ultra-premium, modern, mobile-first eCommerce website for Toy Shopee, a toy and gift store located in Badlapur, Maharashtra, India.

The website must look comparable to premium brands such as Hamleys, FirstCry, Lego, and Disney Store while maintaining local business trust and accessibility.

Technology Stack:

* Next.js 15+
* React 19+
* TypeScript
* Tailwind CSS
* Shadcn UI
* Framer Motion
* Zustand
* React Query / TanStack Query
* PostgreSQL
* Prisma ORM
* NextAuth
* Razorpay
* Google Maps API
* Cloudinary
* Vercel Deployment
